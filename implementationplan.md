## 1. رؤية المنتج

**"يلا حجز"** ليس مجرد تطبيق لحجز الملاعب — بل **منصة رياضية اجتماعية متكاملة** تخدم السوق المصري والعربي:

| الجانب | الوصف |
|---|---|
| 🏟️ **حجز الملاعب** | كرة قدم – بادل – بادبول — بحث جغرافي وتأكيد فوري |
| ⚽ **المباريات المفتوحة** | تكوين فرق + الانضمام لمباريات + تقسيم التكلفة تلقائيًا |
| 🎓 **الأكاديميات** | جلسات تدريبية للمدربين واللاعبين |
| 📊 **لوحات تحكم** | لأصحاب الملاعب (إدارة + أرباح) + للـ Admin (صلاحيات + إعلانات) |
| 💰 **تسييل ذكي** | عمولات + إعلانات + ملاعب مميزة + اشتراكات Premium |
| 🔒 **حماية الحجوزات** | Atomic Transactions لمنع الحجز المزدوج + Realtime updates |

## 2. Tech Stack

### تطبيق الموبايل (Android + iOS)

| المكون | التقنية | السبب |
|---|---|---|
| Framework | **React Native + Expo SDK** | كود واحد للمنصتين + سرعة تطوير |

## 4. أنواع المستخدمين والصلاحيات

### 👤 Player (لاعب)
- البحث عن الملاعب والحجز
- إنشاء مباراة أو الانضمام لمباراة مفتوحة
- التسجيل في جلسات تدريبية
- التقييم والمراجعة بعد الحجز
- إضافة ملاعب للمفضلات
- الدفع الإلكتروني أو الكاش
- إلغاء الحجز (حسب سياسة الإلغاء)

### 🏢 Owner (صاحب ملعب)
- إضافة ملاعب جديدة (تحتاج موافقة Admin)
- رفع صور وتحديد المرافق
- إدارة الجدول الأسبوعي (الأوقات المتاحة والأسعار)
- حظر أوقات معينة يدويًا
- متابعة الحجوزات والأرباح
- الرد على التقييمات
- طلب تسوية الأرباح

### 🛡️ Admin (إدارة المنصة)
- الموافقة على/رفض أصحاب الملاعب والملاعب الجديدة
- إدارة جميع المستخدمين والصلاحيات
- إدارة الإعلانات (إنشاء + تعديل + تتبع)
- تقارير مالية شاملة (العمولات + التسويات)
- إدارة البلاغات والشكاوى
- إعدادات المنصة (نسبة العمولة، سياسة الإلغاء)
- سجل التدقيق (Audit Log)

### 🎓 Coach (مدرب) — امتداد لدور Owner
- إنشاء أكاديمية
- إضافة جلسات تدريبية
- متابعة المسجلين والأرباح

## 5. الميزات الأساسية

### 5.1 استكشاف وحجز الملاعب ⭐

| الميزة | التفاصيل |
|---|---|
| البحث الجغرافي | خريطة تفاعلية + "الأقرب لي" باستخدام PostGIS |
| الفلترة المتقدمة | حسب الرياضة، المنطقة، السعر، نوع الأرضية، الحجم، التقييم |
| تفاصيل الملعب | صور (gallery)، مرافق، تقييمات، موقع على الخريطة |
| الأوقات المتاحة | عرض لحظي (Realtime) مع تحديث فوري عند الحجز |
| الحجز الآمن | Atomic Transaction — `book_slot()` function |
| كود الحجز | كود فريد مكون من 6 أحرف لتأكيد الحجز |
| الإلغاء | حسب سياسة الإلغاء (ساعتين قبل الموعد) |

### 5.2 المباريات المفتوحة والفرق ⚽

| الميزة | التفاصيل |
|---|---|
| إنشاء مباراة | تحديد الملعب + الوقت + عدد اللاعبين المطلوبين |
| الانضمام | أي لاعب يقدر ينضم لمباراة مفتوحة (public) |
| تقسيم التكلفة | `cost_per_player = total_price / required_players` تلقائيًا |
| التتبع | حالة المباراة (open → full → in_progress → completed) |
| الإشعارات | تنبيه اللاعبين عند اكتمال الفريق أو إلغاء المباراة |

### 5.3 الأكاديميات والجلسات التدريبية 🎓

| الميزة | التفاصيل |
|---|---|
| إنشاء أكاديمية | اسم + وصف + لوجو + نوع الرياضة |
| جلسات تدريبية | تاريخ + وقت + ملعب + سعر + حد أقصى للمتدربين + مستوى |
| التسجيل | اللاعب يسجل مباشرة + دفع |
| الإدارة | المدرب يتابع المسجلين والأرباح |

### 5.4 لوحة تحكم صاحب الملعب 📊

| الميزة | التفاصيل |
|---|---|
| إضافة ملعب | معلومات + صور + مرافق + موقع جغرافي |
| الجدول الأسبوعي | تكرار تلقائي (Template) — لا حاجة لإدخال كل يوم يدويًا |
| إدارة الحجوزات | اليومية والشهرية + فلترة حسب الحالة |
| الأرباح | تقارير مفصلة (إجمالي + عمولة المنصة + صافي) |
| التقييمات | عرض + الرد على تقييمات اللاعبين |
| حظر أوقات | قفل أوقات معينة يدويًا (صيانة، حدث خاص) |

### 5.5 لوحة تحكم الإدارة 🛡️

| الميزة | التفاصيل |
|---|---|
| إدارة المستخدمين | عرض + تعديل الصلاحيات + إيقاف/تفعيل |
| الموافقة على الملاعب | مراجعة الملاعب الجديدة + قبول/رفض |
| الإعلانات | إنشاء + تعديل + تتبع (impressions, clicks) + استهداف جغرافي |
| التقارير المالية | إجمالي الحجوزات + العمولات + التسويات |
| البلاغات | مراجعة + حل الشكاوى |
| الإعدادات | نسبة العمولة + سياسة الإلغاء + حدود النظام |
| سجل التدقيق | كل عملية CRUD مسجلة مع المستخدم والوقت |

## 6. نظام الربح والتسييل

### 6.1 عمولة على كل حجز 💰
- نسبة افتراضية: **7%** من قيمة الحجز (قابلة للتعديل من الإعدادات)
- يتم حسابها تلقائيًا داخل `book_slot()` function
- `platform_fee` يُحسم من `total_price` قبل تحويل `owner_amount`

### 6.2 إعلانات مدفوعة 📢
| المكان | النوع |
|---|---|
| الصفحة الرئيسية | بنرات إعلانية كبيرة |
| نتائج البحث | إعلانات بين النتائج |
| صفحة تفاصيل الملعب | إعلانات جانبية |
| Interstitial | شاشة إعلان كاملة (اختياري) |

- تتبع المشاهدات (impressions) والنقرات (clicks)
- استهداف جغرافي (حسب المدينة) + حسب نوع الرياضة

### 6.3 ملاعب مميزة (Featured Pitches) ⭐
- صاحب الملعب يدفع مقابل ظهور ملعبه في أعلى نتائج البحث
- `is_featured = true` + `featured_until` لتحديد فترة الظهور

### 6.4 اشتراكات Premium (مرحلة لاحقة) 👑
- خصومات على الحجوزات
- أولوية في الحجز
- إحصائيات متقدمة للاعبين
- مباريات حصرية

## 11. استراتيجية الأمان

| المستوى | الحماية |
|---|---|
| **Auth** | Supabase Auth — Phone OTP + OAuth + JWT |
| **Database** | RLS على كل جدول |
| **API** | anon key (read) + service_role (Edge Functions فقط) |
| **Payments** | Idempotency keys + Webhook verification |
| **Booking** | `SELECT ... FOR UPDATE` — Atomic transaction |
| **Files** | Storage مع سياسات محددة لكل bucket |
| **Data** | Soft delete للبيانات المالية |
